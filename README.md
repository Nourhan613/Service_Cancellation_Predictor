# Service Cancellation Predictor
This project detects which customers are likely to leave a service or cancel a subscription to a service.
## How the project works is as follows:
1. Preprocessing, Data Cleaning
2. Algorithms
   - Logistic Regression
   - SVM
   - Decision Tree
3. GUI
